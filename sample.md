# SimpleMD Test Document

Welcome to **SimpleMD** - a clean and modern Markdown viewer for Windows!

## Features

- **Live Preview** - See your markdown rendered in real-time
- **Syntax Highlighting** - Code blocks with proper highlighting
- **Dark Mode** - Easy on the eyes
- **Export to HTML** - Share your documents
- **Export to PDF** - Create professional PDFs with customizable settings
- **Print Support** - Print your markdown documents

## Markdown Examples

### Text Formatting

- **Bold text** using `**bold**`
- *Italic text* using `*italic*`
- ~~Strikethrough~~ using `~~strikethrough~~`
- `Inline code` using backticks

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
